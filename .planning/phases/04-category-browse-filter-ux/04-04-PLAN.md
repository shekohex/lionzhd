  - `filters: { category: string | null }`
- Remove the artificial `setTimeout` loading state from both pages.
- Add a navigation-driven `isSwitchingCategory` state in each page:
  - Set true in `router.visit` `onStart` for category changes, and false in `onFinish`/`onError`.
  - While switching, replace the current grid with skeleton cards.
- Implement category selection behavior:
  - URL source of truth: use `router.visit(route('movies', { category }))` / `route('series', { category })`.

  - Selecting active category toggles back to All.
  - Drop pagination to page 1 on category change by not carrying `page`.
  - `preserveState: true` (mobile sheet stays open), `preserveScroll: false`, and call `scrollToTop('instant')`.
  - Use partial reloads:
    - Movies category change: `only: ['movies', 'filters']`
    - Series category change: `only: ['series', 'filters']`
- Update empty states:
  - When a category is selected and total is 0: show an empty-category prompt encouraging trying another category + a CTA to reset to All.
  - Preserve existing All-categories empty state copy.
- Keep infinite scroll behavior for mobile (existing `useInfiniteScroll`) and ensure category changes reset to page 1 and show fresh data.
  </action>
