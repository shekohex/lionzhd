- Keep desktop sidebar markup/behavior unchanged.

- Mobile sheet: switch SheetContent from left slide-over to bottom sheet.
  - Change `side="left"` to `side="bottom"`.

    - full width (`w-full max-w-none`)

    - rounded top corners (`rounded-t-xl`)

  - Wrap the list container in a flex child with scroll (e.g. `className="flex-1 overflow-y-auto px-4 pb-6"`).
  - Keep `SheetHeader` + `SheetTitle` visible above the scroll area.

- Close the mobile sheet on any successful category selection (including All + toggling the current category back to All):
  - Add a small helper (e.g. `selectCategoryAndClose(nextCategory)`) that calls `setIsMobileSheetOpen(false)` then calls `onSelectCategory(nextCategory)`.
  - Use this helper for:
    - All categories option
    - Each enabled category option
  - Do NOT close the sheet for disabled categories (they should remain disabled with tooltip).
