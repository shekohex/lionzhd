  - Auto-load near bottom.
  - No chain-loading: only trigger loadMore on a false→true near-bottom transition.

- Ensure deep links like `?page=3` behave sensibly without chain-loading:
  - Initial `allData` is whatever page the server rendered (do not auto-backfill pages 1..2).
  - Subsequent loadMore appends page 4+ normally.
